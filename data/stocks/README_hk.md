# 港股数据导入说明

## 当前状态

由于网络原因，港股数据暂时无法从东方财富 API 下载。

## 手动创建港股数据文件

如需导入港股数据，请按以下步骤操作：

### 1. 创建港股 CSV 文件

创建文件 `data/stocks/hk_stocks.csv`，格式如下：

```csv
code,name,market,type
0001,长和,HK,港股
0002,中电控股,HK,港股
0003,香港中华煤气,HK,港股
0005,汇丰控股,HK,港股
0006,电能实业,HK,港股
...
```

### 2. 运行导入脚本

```bash
python scripts/import_stocks_to_db.py
```

## 港股数据源

### 方法 1: 东方财富网
访问：http://quote.eastmoney.com/center/gridlist.html#hk

### 方法 2: 港交所披露易
访问：https://www.hkex.com.hk/Market-Data/Securities-Prices/Equities-Quote

### 方法 3: 腾讯财经
访问：https://stockapp.finance.qq.com/mstats/#

## 港股行业分类

港股行业分类使用简化版，根据股票名称关键词自动判断：
- 金融：银行、证券、保险、金融、信托
- 房地产：地产、置业、物业
- 科技：科技、软件、信息、网络、智能、芯片、半导体
- 医药：医药、药业、生物、医疗、健康
- 消费：消费、食品、饮料、餐饮、零售、服装
- 工业：工业、制造、机械、设备、工程
- 能源：能源、石油、天然气、化工、资源
- 电信：通信、电信、移动
- 公用事业：电力、燃气、水务
- 交通运输：运输、物流、港口、航运、航空
- 互联网：互联网、游戏、电商
