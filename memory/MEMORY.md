## 核心功能
1. **数据采集**: 新闻 (东方财富/新浪/财联社)、行情 (Baostock 为主，AkShare 为备用)、资金流
2. **分析模块**: NLP 情感分析、技术分析 (MA/MACD/RSI/KDJ/布林带/OBV/BIAS/VR)、资金分析、波动率分析 (ATR)
3. **信号融合**: 加权评分，支持动态权重调整
4. **市场状态判断**: 牛市/熊市/震荡市识别（基于沪深 300 指数）
5. **板块联动分析**: 行业/概念板块资金流向与联动性评分
6. **决策引擎**: 仓位管理、止盈止损
7. **风险管理**: 黑名单、ST 过滤、组合级风控（行业集中度≤30%、最大回撤 15%）
8. **Web 面板**: Streamlit 可视化，端口 8701
9. **通知系统**: 钉钉机器人（带签名）、微信机器人、Webhook
