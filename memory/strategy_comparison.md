## 新策略优势

1. **收益率大幅提升**：74% → 157%，翻倍有余
2. **盈亏比更高**：3.07 → 3.58，赚得更多亏得更少
3. **平均盈利提升**：11% → 18%，单笔盈利能力强
4. **交易频率降低**：174 笔 → 123 笔，减少 29%
5. **夏普比率更高**：风险调整后收益更好

## 新策略劣势

1. **胜率略降**：48.3% → 46.3%
2. **回撤增大**：8.8% → 14.8%
3. **平均亏损增加**：3.6% → 5.1%

## 结论

**新策略 V2 是更优选择**，适合追求长期稳定盈利的投资者：

- 收益率翻倍是核心优势
- 盈亏比 >3.5，符合稳定盈利标准
- 交易成本更低（减少 29% 交易次数）
- "少而精"的交易风格

需要注意：回撤较大，需要更好的心理承受能力。
