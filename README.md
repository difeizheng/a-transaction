# A 股自动监控与交易系统

一套基于 Python 的 A 股市场自动监控与交易信号生成系统。

## 功能特点

- **多维度数据采集**: 新闻、行情、资金流向
- **NLP 情感分析**: 自动分析新闻利好/利空
- **技术指标分析**: MA、MACD、RSI、KDJ、布林带
- **资金流向分析**: 北向资金、主力资金监控
- **信号融合引擎**: 多因子加权综合评分
- **智能决策**: 自动生成买卖信号和仓位建议
- **风险控制**: 止损止盈、仓位管理、黑名单

## 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                     A 股自动监控系统                      │

│      ↓       │     数据处理层 (NLP/技术面/资金面)        │

│  输出层    │  信号输出  │  实时监控  │  通知推送       │

```

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置系统

编辑 `config.yaml` 文件，配置以下内容：

- 股票池设置（沪深 300/自定义）
- 监控频率
- 交易参数（止盈止损、仓位管理）
- 通知方式（微信/钉钉）

### 3. 运行系统

```bash
# 持续监控模式
python main.py

# 执行一次
python main.py --once
```

## 信号评分算法

```
综合得分 = 新闻分 × 0.35 + 技术分 × 0.30 + 资金分 × 0.25 + 情绪分 × 0.10

信号判定:
- 综合得分 >= 0.7: 强烈买入
- 综合得分 >= 0.5: 买入
- 综合得分 >= -0.3: 持有
- 综合得分 >= -0.6: 卖出
- 综合得分 < -0.6: 强烈卖出
```

## 模块说明

### 数据采集 (collectors)
- `news_collector.py`: 新闻采集（东方财富、新浪、财联社）
- `price_collector.py`: 行情数据（AkShare）
- `fund_collector.py`: 资金流向（北向资金、主力）

### 分析器 (analyzers)
- `sentiment_analyzer.py`: NLP 情感分析
- `technical_analyzer.py`: 技术指标分析
- `fund_analyzer.py`: 资金流向分析

### 引擎 (engine)
- `signal_fusion.py`: 多因子信号融合
- `decision_engine.py`: 交易决策生成
- `risk_manager.py`: 风险管理
